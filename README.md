## 프로젝트 개요
- 2022년부터 계속 된 주요 금리 인상으로 인해 부동산 대란이 계속 되고 있습니다.
- 같은 시기에 많은 이들이 정보 부족으로 인해 지속적인 손해를 보았습니다.
- 저희는 기존의 부동산 어플과 사이트가 제공하지 않는 양질의 데이터를 소비자들에게 제공하기 위해 프로젝트를 시작하게 되었습니다.
